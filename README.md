# О проекте

В данном репозитории собраны руководства с пошаговыми инструкциями по самостоятельному обслуживанию мотоциклов [Royal Enfield](https://www.royalenfield.com).

## Зачем это нужно?

- **Самостоятельное обслуживание**: экономия на сервисе и понимание устройства собственного мотоцикла.
- **Достоверность**: материалы основаны на публичных источниках, мануалах и опыте владельцев.
- **Удобство**: все руководства структурированы, доступны в виде сайта и могут дополняться любым желающим.

### Кому это будет полезно?

- Владельцам [Royal Enfield](https://www.royalenfield.com), желающим разобраться в мотоцикле и проводить обслуживание своими руками.
- Новичкам, которые ищут понятные пошаговые инструкции.
- Опытным механикам, готовым делиться знаниями и помогать сообществу.

## Устройство репозитория

- Все заметки хранятся в папке `content/` в формате **Markdown**. Директория непосредственно внутри `content/` называется *модулем* и содержит заметки по одной или нескольким (группе) концептуально схожих моделях мотоциклов RE.
- Для генерации сайта используется генератор статических сайтов [Quartz](https://quartz.jzhao.xyz/).

## Работа с локальной копией

Чтобы работать с материалами локально, вам потребуется скопировать репозиторий и настроить окружение для генерации сайта.

### Предварительные требования

- Установленный [Node.js](https://nodejs.org/) (версия 18 или выше).
- [Git](https://git-scm.com/book/ru/v2/%D0%92%D0%B2%D0%B5%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5-%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0-Git).

### Инструкция

1. **Клонирование репозитория**
   Выполните команды или скачайте репозиторий в виде zip-архива:
	```bash
	git clone https://github.com/watisluv1910/re-diy-maintenance.git
	cd re-diy-maintenance
	```
2. **Установка зависимостей**
   Репозиторий использует [Quartz](https://github.com/jackyzha0/quartz) для генерации статического сайта. 
   Установите необходимые пакеты, выполнив команду:
	```bash
	npm install
	```
3. **Запуск локального сервера**
   Выполните в корневой директории репозитория:
	```bash
	npx quartz dev
	```
	После выполнения команды сайт будет доступен по адресу `http://localhost:8080`.  Сервер автоматически отслеживает изменения в папке `content/` и обновляет страницу.
4. **Просмотр и редактирование**.
   Все заметки находятся в каталоге `content/`. Для редактирования заметок подойдёт любой текстовый или Markdown редактор.
