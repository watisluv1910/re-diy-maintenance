# Описание раздела

В данном разделе собраны подробные пошаговые инструкции по выполнению операций по самостоятельному обслуживанию #classic-350-reborn. Инструкции были подготовлены на основе сборника [All New Classic 350 DIY Videos](https://youtube.com/playlist?list=PLT1BEDU54E1QNhNm0RBd0BGXzAH1leldc&si=_gbwqCGGBN8w6IeR) с официального [канала Royal Enfield](https://www.youtube.com/@royalenfield) на YouTube. В качестве дополнительной информации в инструкциях приводятся сведения из [[../owners-manual-re-classic-350-reborn.pdf|мануала]] и из различных форумов владельцев RE.

Если инструкции на основе определённого видео из сборника отсутствуют в данном разделе, значит, либо видео было опубликовано недавно и ещё не было преобразовано в пошаговую инструкцию, либо видео не несёт в себе достаточной практической ценности для выполнения такого преобразования.

Также в разделе могут присутствовать инструкции по выполнению операций, отсутствующих в вышеназванном сборнике.

## Тэги

Все заметки в данном разделе имеют обязательный тэг *обслуживание*. 

Тэг модели также обязателен. Если  может быть выполнена для любой модели RE с мотором #J-серия, то в качестве тега модели должен использоваться тег #J-серия. Добавление тега конкретной модели (например, #classic-350-reborn) в таком случае является опциональным.

## Структура заметки

Каждая заметка в разделе состоит из следующих частей, всегда расположенных в следующем порядке:
1. [[#Раздел "Требуемые инструменты"|Требуемые инструменты]] (опционально).
2. [[#Раздел "Требуемые расходники"|Требуемые расходники]] (опционально).
3. [[#Раздел "Ограничения"|Ограничения]] (опционально).
4. [[#Раздел "Порядок действий"|Порядок действий]].
5. Ссылка на видео из плейлиста [All New Classic 350 DIY Videos](https://youtube.com/playlist?list=PLT1BEDU54E1QNhNm0RBd0BGXzAH1leldc&si=_gbwqCGGBN8w6IeR) или на любой другой ресурс, на основе которого составлена инструкция, если таковой имеется (опционально).

### Раздел "Требуемые инструменты"

Содержит список всех инструментов, необходимых для выполнения операции. Если описание используемого [[../Инструменты/index#Определение инструмента|инструмента]] представлено в [[../Инструменты/index|разделе "Инструменты"]], на него будет предоставлена ссылка. Также, если количество требуемых идентичных инструментов `N > 1`, оно будет явно указано после наименования инструмента в формате `<наименование> x N`.

### Раздел "Требуемые расходники"

По аналогии с разделом [[#Раздел "Требуемые инструменты"|Требуемые инструменты]], содержит список всех [[../Расходники/index#Определение расходника|расходников]], необходимых для выполнения операции. Если описание используемого расходника представлено в [[../Расходники/index|разделе "Расходники"]], на него будет предоставлена ссылка. Также, если количество требуемых идентичных расходников `N > 1`, оно будет явно указано после наименования расходника в формате `<наименование> x N`.

### Раздел "Ограничения"

**!** Обязателен к прочтению **ПЕРЕД** выполнением операции.

Предоставляет информацию о важных моментах, на которые стоит обратить внимание при выполнении инструкции из [[#Раздел "Порядок действий"|раздела "Порядок действий"]].

### Раздел "Порядок действий"

Основной раздел заметки. Содержит в себе пошаговую инструкцию по выполнению операции. Дополнительные сведения, касающиеся одного или нескольких шагов инструкции (например, степень затяжки болтов) могут быть определены в конце данного раздела.
