# Доливка моторного масла

## Причины проведения

- [[Проверка уровня моторного масла]] выявила несоответствие целевому показателю.
- Проводится [[Замена моторного масла]].

## Требуемые инструменты

- [[../Инструменты/Воронка для технических жидкостей|Воронка для технических жидкостей]].
- [[../Инструменты/Мерный пластиковый стакан|Мерный пластиковый стакан]].

## Требуемые расходники

- [[../Расходники/Моторное масло|Моторное масло]].

## Ограничения

1. Никогда не заливать масло выше максимального уровня.

## Порядок действий

1. Провести [[Проверка уровня моторного масла|проверку уровня масла]], если есть необходимость.
2. Открыть крышку заливной горловины:
   ![[attachments/oil-cap.png]]
3. Вставить [[../Инструменты/Воронка для технических жидкостей|воронку]] и аккуратно наливать масло из [[../Инструменты/Мерный пластиковый стакан|стакана]], пока уровень не достигнет максимума:
   <div style="display: flex; gap: 2%">
	   <img src="attachments/oil-filling.png" style="width: 49%"/>
	   <img src="attachments/oil-level.png" style="width: 49%"/>
   </div>
4. Закрутить крышку заливной горловины.
