# Описание раздела

В данном разделе собрана информация о внешнем виде, свойствах и местах возможной покупки большинства инструментов, которые могут пригодиться при обслуживании #classic-350-reborn. 

Если какие-то инструменты отсутствуют в данном разделе - значит они не стоят отдельной заметки (как пример, рожковый гаечный ключ).

Вне зависимости от факта наличия заметки об инструменте в данном разделе, если инструмент будет необходим для выполнения конкретной операции по обслуживанию мотоцикла, он будет перечислен в разделе **'Требуемые инструменты'** для заметки по данной операции. Подробнее о структуре заметок по обслуживанию в [[../Обслуживание/index|описании соответствующего раздела]].

## Тэги

Все заметки в данном разделе имеют обязательный тэг *инструменты*. Если инструмент не специфичен по отношению к модели #classic-350-reborn или, чаще всего, любой другой модели RE с мотором #J-серия, то тег модели к заметке не добавляется.

## Структура заметки

Каждая заметка в разделе состоит из следующих частей:
- [[#Раздел "Свойства"|Свойства]].
- [[#Раздел "Внешний вид"|Внешний вид]].
- [[#Раздел "Где купить"|Где купить]] (опционально).

### Раздел "Свойства"

Содержит информацию о некоторых характеристиках инструмента и/или нескольких его версий. Возможные варианты свойств для определения:
- `Поставляется с мотоциклом` (да/нет) – определяет, поставляется ли инструмент в наборе в комплекте с мотоциклом. **Обязательно** к определению.
- `Предназначение` – для чего необходим инструмент, базовые сценарии его использования.
- `Тип` – возможные разновидности инструмента, через `/`.

Дополнительные свойства могут быть определены и должны быть расположены после вышеперечисленных.

### Раздел "Внешний вид"

Содержит одно (или более) изображение инструмента.

### Раздел "Где купить"

Содержит ссылки на онлайн-магазины (чаще всего маркетплейсы), в которых можно приобрести инструмент, описываемый в записке. Данные ссылки даются для общего представления, т.е. могут не являться предложением на рынке. Они должны выступать как отправная точка для последующего поиска устраивающего конкретно Вас варианта на конкретно Вам удобной платформе.
